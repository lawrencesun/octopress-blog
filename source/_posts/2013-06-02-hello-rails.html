两天时间完成了codeschool的Rails for Zombies，虽然还有不少内容似似而非，练习也基本是依样画瓢，不过总算对rails有了个初步印象。加上昨天用railsinstaller在windows部署环境时，接触了rails的建立。

晚上试着根据railsguide在ubuntu用rails建立个博客，结果出现了问题。

在执行

$ rails server

出现了 runtime...错误。

在ruby-china搜索到了答案， 

gem install execjs
gem install therubyracer
sudo apt-get install nodejs
不过在执行第二行时，又出现了

require ref...libv8...的错误

在stackoverflow搜到解决方法：

gem install therubyracer --version=0.10.2
